Predict Next Words

This is the capstone project of Data Science Specialization on Coursera offered by Johns Hopkins University. In this project we build a web application to predict the next words following each key press. This kind of applications have been widely used for text input in mobile devices such as cell phones and tablets.

Shiny: click on one figure to get another

A common task in interactive plotting is to click on figures to extract more information. Shiny is a handy tool for this kind of work. This interactive document demonstrates and explains how to click on a figure to get another using the data from the clicked location.

Shiny: click on a map

In this tutorial, we will continue the discussion of Shiny click with one more example: click on a map to get something else. The purpose is to further understand the click and its application. Shiny click provides a flexible tool for interactive plotting with map.
